# Блог H-Studio Business

Этот каталог содержит статьи блога в формате Markdown.

## Структура поста

Каждый пост должен иметь следующий формат frontmatter:

```yaml
---
title: "Название статьи"
slug: "url-slug"
date: "2024-11-28"
author: "H-Studio Team"
category: "Автоматизация"
tags: ["автоматизация", "КП", "расчёты"]
excerpt: "Краткое описание статьи для превью"
---
```

**Примечание:** Поле `image` больше не используется. Обложки генерируются автоматически с чередованием двух цветов бренда.

## Создание нового поста

### Через скрипт

```bash
npm run blog:generate -- --topic "автоматизация КП" --category "Автоматизация" --tags "автоматизация,КП,расчёты"
```

### Вручную

1. Создайте новый файл `.md` в этой папке
2. Используйте slug в качестве имени файла (например, `kak-avtomatizirovat-kp.md`)
3. Добавьте frontmatter с метаданными
4. Напишите контент в формате Markdown

## GitHub Actions

Для автоматической генерации постов используйте GitHub Actions workflow:
`.github/workflows/generate-blog-post.yml`

Запустите workflow через GitHub UI, указав тему и опциональные параметры.
