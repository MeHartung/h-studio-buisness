## Настройка переменных окружения

### Локально

1. Скопируйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Заполните переменные в `.env`:
   ```env
   NEXT_PUBLIC_SITE_URL=https://www.h-studio-tech.ru
   OPENAI_API_KEY=your_key_here  # опционально
   ```

### В GitHub

1. Перейдите в Settings → Secrets and variables → Actions
2. Добавьте секрет `OPENAI_API_KEY` (если планируете использовать AI-генерацию)

## Формат даты в постах

Дата в frontmatter должна быть в формате `YYYY-MM-DD`:
```yaml
date: "2024-11-28"
```

Если дата отсутствует или неверная, система автоматически использует текущую дату.
